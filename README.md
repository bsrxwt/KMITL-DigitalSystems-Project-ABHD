## 🎮 About The Project
โปรเจกต์ชุบชีวิตเกมอาร์เคดคลาสสิกทำลายอิฐ (Brick Breaker) ด้วยการออกแบบวงจรดิจิทัลและประมวลผลระดับฮาร์ดแวร์บนบอร์ด FPGA พร้อมแสดงผลกราฟิกออกสู่หน้าจอ VGA แบบเรียลไทม์

### ✨ Features
* **Single Player & Multi Player:** รองรับทั้งการเล่นคนเดียว (ท้าทายด้วยหัวใจ 1 ดวง) และโหมดแท็กทีม
* **Adjustable Difficulty:** ปรับความเร็วลูกบอลได้ 3 ระดับ (Easy, Normal, Hard) ผ่านสวิตช์บนบอร์ด
* **Hardware Touch Control:** ควบคุมแป้นรับลูกบอล (Paddle) ซ้าย-ขวา ตอบสนองรวดเร็วผ่าน Push Buttons
